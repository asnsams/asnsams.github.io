FE-ATnet: Chest Radiograph Interpretation Using Feature Enhanced Multi-stage Attention 

 Course No: EEE 426 (​​​Biomedical Signals Instrumentation and Measurement Laboratory)
 Platform: Python
 Description:    Chest radiography is a typical diagnostic radiology test used to detect and diagnose thoracic diseases. However, detecting the presence of multiple diseases using chest X-rays remains a difficult task. To predict the risk of five prevalent thoracic diseases, this paper describes a supervised multi-label classification system using deep convolutional neural networks (CNNs). We propose a novel feature enhanced multi stage attention network with pre-trained Densenet121 as backbone. We also present a label smoothing technique for dealing with unlabeled chest x-ray samples and investigate various pooling combinations. On the independent test set for the CheXpert competition, which consisted of 500 CXR experiments, the proposed approach achieved a mean AUC of 0.907. It is by far the highest single model mean AUC score in the CheXpert competition that we are aware of.

IOT weather sensor (Arduino and STM32cube Implementation)

 Course No: EEE 416 (​Microprocessor and Embedded System Laboratory )
 Platform: Proteus, Arduino, STMCUBE
 Description:   ​The main goal of this project is to design and build an effective monitoring system in which the required parameters are monitored remotely over the internet, the data collected from the sensors is stored in the cloud, and the estimated trend is projected on a mobile device. TETRIS GAME IMPLEMENTATION

 Course No: EEE 304 (Digital Electronics Laboratory)
 Platform: Verilog, Proteus & Arduino
 Description:  The objective of TETRIS is to use the pieces to create as many horizontal lines of blocks as possible. When a line is completed, it disappears, and the blocks placed above fall one rank. The game ends when the playing field is filled. The longer the player can delay this inevitable outcome, the higher their score will be. We implemented the game using two different method. First, we used verilog to code the game. And then we use arduino and proteus to code and simulate the game virtually.

ANALYSIS OF HVDC SYSTEM

  Course No: EEE 306 (Power System I Laboratory) 
  Platform: Simulink
  Description:  The project illustrates the Simulink modelling of a High Voltage DC Transmission link using 12pulse based convertors between a 500kV, 5000MVA, 60Hz, system to a 345kV, 10000MVA, 50Hz system over a 1000MW(500kV, 2kV) DC interconnections. Through this model, we analyzed -
1) Real time response/ Transient and Steady state response 2) Frequency response of the AC system 3) Comparison between Distributed line parameter and Pi section line parameter through Real time and frequency response. 4) DC line fault 5) AC line to ground fault

PHONEME BASED BANGLA DIGIT RECOGNITION

 Course No: EEE 312 (Digital Signal Processing I Lab)
 Platform: Matlab
 Description:  This research work aims at recognizing the Bangla digit from 0-9 by separating the phonemes of the digits. The ambiguity in phonemes in Bangla speech is more extreme and varied than that of English speech since Bangla stems from the “Indo-European language family”. In this research, dataset is collected in manual fashion. Then, the digits are broken into their respective phonemes (both voiced and unvoiced part). Next, Mel frequency Cepstral Coefficients (MFCC) features of the segmented digits are calculated and trained using artificial neural network, which is used to recognize the unknown digit. The proposed system is implemented through MATLAB and accuracy of the system averages above 90%.

BANGLA NUMBER PLATE RECOGNITION USING TEMPLATE MATCHING

 Course: EEE 212 Numerical Technique Laboratory
 Platform: Matlab
 Description:  In this project, we used template image processing and matching technique for the recognition of the characters from the license number plate. In this method , first the image of a car license number plate is taken as input, then pre-processing steps such as conversion to Gray-scale image, dilation, erosion, convolution is done to remove noise from the input image. Then each character in the number plate is segmented using suitable filter. After segmentation, recognition of characters is done by matching templates in the database to the segmented characters.
