 Course No: EEE 304 (Digital Electronics Laboratory)
 Platform: Verilog, Proteus & Arduino

  Platform: Simulink

PHONEME BASED BANGLA DIGIT RECOGNITION

 Course No: EEE 312 (Digital Signal Processing I Lab)
 Platform: Matlab
 Description:  This research work aims at recognizing the Bangla digit from 0-9 by separating the phonemes of the digits. The ambiguity in phonemes in Bangla speech is more extreme and varied than that of English speech since Bangla stems from the “Indo-European language family”. In this research, dataset is collected in manual fashion. Then, the digits are broken into their respective phonemes (both voiced and unvoiced part). Next, Mel frequency Cepstral Coefficients (MFCC) features of the segmented digits are calculated and trained using artificial neural network, which is used to recognize the unknown digit. The proposed system is implemented through MATLAB and accuracy of the system averages above 90%.

BANGLA NUMBER PLATE RECOGNITION USING TEMPLATE MATCHING

 Course: EEE 212 Numerical Technique Laboratory
 Platform: Matlab
 Description:  In this project, we used template image processing and matching technique for the recognition of the characters from the license number plate. In this method , first the image of a car license number plate is taken as input, then pre-processing steps such as conversion to Gray-scale image, dilation, erosion, convolution is done to remove noise from the input image. Then each character in the number plate is segmented using suitable filter. After segmentation, recognition of characters is done by matching templates in the database to the segmented characters.
